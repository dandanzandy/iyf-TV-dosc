技术栈

爱壹帆影视站采用前沿技术，保证高效、稳定和可扩展性。

  • Vue3：构建响应式前端，简化开发流程。
  • TypeScript：确保类型安全，减少开发中的错误。
  • Node.js：高效后端，支持流媒体处理。
  • Vite：极速开发，支持热更新。
  • Pinia：轻量级状态管理工具，简化全局状态管理。
  • Vue Router：支持SPA，流畅的用户体验。

项目特性

模块化设计

项目采用模块化设计，方便功能扩展和自定义。

极速播放

无广告、极速加载，提供最流畅的播放体验。

响应式布局

支持桌面与移动端设备，优化用户体验。

类型安全

使用 TypeScript 保证前后端交互的类型安全。

安装与使用

  1. 克隆项目： 使用 Git 克隆项目：
    git clone https://github.com/your-username/iyfyy-tv.git
  2. 安装依赖： 安装项目依赖：
    npm install
  3. 启动开发环境： 启动开发服务器：
    npm run dev
  4. 构建生产版本： 构建生产版本：
    npm run build
